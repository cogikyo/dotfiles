# review

Review code changes or code areas with explicit scope control, focused criticism, and safe fix follow-up.

## Commands

- `/review` - Infer review scope, ask only when ambiguous, then review
- `/review orchestrate` - Manage a broad review using focused passes when available
- `/review fix` - Apply approved fixes from review findings
- `/review debugger` - Correctness-focused review
- `/review auditor` - Safety and blast-radius review
- `/review profiler` - Performance and cost review
- `/review janitor` - Cleanliness and coupling review
- `/review architect` - Big-picture clarity review
- `/review modernize` - Modernization review
- `/review simplify` - Simplicity and complexity review
- `/review scribe` - Documentation and comment review

Read `INSTRUCTIONS.md` for full guidance.
